# 🦆 Качка [Duckloader]

**YouTube загрузчик с графическим интерфейсом**

версия 0.1 — сонная утка

!Скриншот - screenshot.png
---

## ✨ Возможности

- Скачивание видео с YouTube в 5 качествах (360p → лучшее)
- Скачивание только аудио (MP3)
- Одновременная загрузка до 4 видео
- Выбор папки для сохранения
- Лог загрузок
- Красивый интерфейс с фоновым изображением
- Звук кряканья после загрузки 🦆

---

## 📥 Установка и запуск

### 1. Установи зависимости

bash
sudo apt install yt-dlp ffmpeg
2. Скачай файлы
Скачай все файлы из этого репозитория или клонируй:

bash
git clone https://github.com/VibeCoder0000/duckloader.git
cd duckloader
3. Запусти программу
bash
./duckloader
4. Сборка из исходников (если хочешь)
bash
gcc main.c -o duckloader `pkg-config --cflags --libs gtk+-3.0`
🎮 Как пользоваться
Вставь ссылки на YouTube в поля (до 4 штук)

Выбери качество видео

Нажми кнопку 📦 качка$ sudo удача 📦

Жди завершения и радуйся 😁
